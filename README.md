# C To-Do List Application

A simple console-based To-Do List application written in C.
C dilinde yazılmış basit bir konsol tabanlı Yapılacaklar Listesi uygulaması.

## Overview / Genel Bakış

This project implements a task management system where users can add, list, and delete tasks. The data is persisted in a text file.
Bu proje, kullanıcıların görev ekleyebileceği, listeleyebileceği ve silebileceği bir görev yönetim sistemi uygular. Veriler bir metin dosyasında saklanır.
